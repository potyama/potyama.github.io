# 使い方

それぞれの空欄にコマ数(分単位のところは分)を入力してください。コマがないところは入力しなくて大丈夫です。

  1. 下にある計算ボタンを押してください
  2. 計算ボタンの下に計算結果が表示されます。

なにか不具合や計算結果が合わない場合はkamikaze[@]gmail[.]comにまでメールを送るか、GitHub (opens new window)にPRを投げてください。
